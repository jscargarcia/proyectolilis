PASOS RÁPIDOS:
--------------------------------------------------------------------------------
1. Copiar proyecto a nueva máquina
2. Instalar Python 3.13+ y MySQL 8.0+
3. Crear BD: CREATE DATABASE empresa_lilis;
4. Editar config/settings.py con credenciales
5. Ejecutar: .\instalar.ps1 (Windows) o ./instalar.sh (Linux)
6. Verificar: python verify_setup.py
7. Iniciar: python manage.py runserver
